{# ================================================================== TAB 3 — Suspected Discontinued ================================================================== #}
⚠️ Suspected Discontinued variants

Variants whose latest CarWale scrape returned not_found — meaning the manufacturer has retired or renamed that trim. These need admin attention. Approving a flag preserves the existing price (so existing owners can still get valuations) and stamps today's date as the last verified price date.

Pending reviews of type discontinued appear in the Review Queue tab. To find new ones, run scrapes via Refresh One on models you suspect have stale variants.
